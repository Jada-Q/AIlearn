# 把设计品味变成可测的科学：一场 20 轮 60 评审的盲测实验

> 2026-06-12 · 从 54 个网站的设计基因库出发，用预注册 A/B 盲测验证理论，再用 AI 评委团把一个页面从 6.5 分磨到 8.0 分的完整记录

"这个设计好不好"通常是一句没法吵赢的话。这次实验试着把它变成可以测量、可以证伪、可以复现的东西。全程一晚上，零真人参与（评委全部是无上下文的 AI agent），但方法论对真人评审同样成立。

---

## 第一步：从素材库归纳理论（不是从书里抄）

起点是一个攒了 54 个网站的设计分析库（每站一张"DNA 卡"：布局/字体/色彩/留白/细节/借鉴警告六个维度）。3 个并行 agent 通读全部卡片做跨站模式统计，合成出一份《网站设计底层逻辑》，核心是 5 条法则：

1. **内容是地基，UI 是托架**（28+ 张卡的警告指向它，零反例）——先审素材，再选骨架
2. **稀缺性 = 力量**——accent 色全站只出手 1-3 次，"频率越低，出现时越像事件"
3. **每个视觉选择都要有叙事理由**——没有理由的效果是事故
4. **站点类型决定骨架**——credibility 站和 conversion 站是两个物种，错配必死
5. **一个 move 是一套组合拳**——不能拆半套用

关键约束：合成时禁止掺入语料外的通用设计知识，所有结论必须附站名和出现次数。封闭语料里的频率数字才有证据效力。

## 第二步：预注册 A/B 盲测（防自己事后找补）

理论是归纳出来的，可能只是这 54 个站的局部风味。于是挑 3 条赌注最大的命题做单变量对照实验：

- 同一个页面做 A/B 两版，**只差被测变量**（如：1 处 accent vs 6 处和谐用彩）
- B 版按"不知道该规则的认真设计师"标准做，不做稻草人
- 评委 prompt 里不放理论，只问"哪版更像高价工作室作品"，顺序随机
- **开测前预注册**：规则版 ≥7/10 票算有效，落盘存档，输了就是输了

结果：3 条命题 30/30 票全过。更有说服力的是评委在没读过理论的情况下自发说出了理论的语言——多彩版被骂 "template trick"，温和放大字号版被骂 "timid"。

意外收获：实验还抓到了**规则打架**——密度脉冲和左对齐纪律冲突时，两张反对票都站对齐。理论因此补了一条边界条款。

## 第三步：自拟考题实战，然后被分数教育

用这套理论从零建一个虚构产品的 landing page（夜间桌面壁纸合集），三个人设的 AI 评委（严苛设计总监 / 挑剔 Mac 用户 / 杂志美术总监）每轮打分 + 给出 2 强 2 弱。

**初版 6.5/10。** 评委的表扬全部落在理论覆盖区（结构、稀缺纪律、metadata 系统），失分全部在理论没写的地方：直引号、基线碰撞、CTA 游离于系统外、没有动态证明。

接下来 20 轮迭代，60 次独立评审，曲线：

```
6.5 → 7.0 → 7.0 → 7.17 → 7.67 → 7.83 → 7.83 → 8.0 → 7.83 → 8.0
→ 7.67 → 7.5 → 7.33 → 7.83 → 7.33 → 7.67 → 7.83 → 7.67 → 7.5 → 8.0（三票一致，收卷）
```

## 三层结论（每层都有票数背书）

| 层 | 分数区间 | 决定因素 |
|---|---|---|
| **理论层** | 0 → 6.5 | 物种/弹药/层级/稀缺四步决策，保证"对" |
| **工艺层** | 6.5 → 7.5 | 弯引号、基线规则、mono 收据层、动态证明，决定"贵" |
| **原创层** | 7.5 → 8.0 | 只属于这个站的系统级 move，评委所谓 "genre 内执行" 和 "studio 作品" 的分界 |

原创层的三个 move（60 次评审零反对票）：

1. **语义化标点**：标题的句号就是全页的 live 数据指示灯，同一个橙点贯穿 ticker、按钮、每条 caption——评委称之为 "punctuation doing a job"
2. **24 小时时间轴**：把一天映射到一张画布的 x 轴，时间戳钉在真实坐标上——评委自己拿尺量完写下 "annotation with integrity"
3. **点阵字标**：footer 大字标连续 4 轮被骂"2023 烂大街"后，改成 live 数据驱动的呼吸点阵——下一轮直接三票 8

## 四个反直觉发现

**1. 真实素材不自动等于更好。** 中途把程序化占位图换成真产品的夜间截图，分数反而从 8.0 掉到 7.5——夜间截图在缩略图尺寸全是黑块。评委的判词值得裱起来："Real stills is a constraint you chose, but art direction of *which* still is still your job."（真实截图是你选的约束，但选哪一帧依然是你的设计工作。）

**2. 评委的批评栏里藏着金点子。** 最大的单轮跃升不是来自我的方案，而是来自评委骂人时顺手给的方子（"让字标自己也用 live 数据画出来"）。具体到可执行的批评是礼物。

**3. 诚实本身能换分。** 把唯一一张静态图老老实实标注 "tonight's still"，评委反馈："labeling the one non-live asset honestly made me believe the others more."（诚实标注那张非实时素材，让我更相信其余素材。）反过来，页面内两个时钟差 46 分钟被评委抓到时，判词是 "fatal to its own conceit"。

**4. 分数会撞测量噪声墙。** 同一个设计选择在不同轮次被不同评委给出相反判决（serif 引言：一轮"自信的收束"，下一轮"borrowed garnish"）。当批评开始互相矛盾、分数振幅小于评委间差异时，继续迭代就是在优化随机数——这时该停手或换维度，而不是再改一轮 CSS。

## 可复用的实验框架（5 步）

1. **归纳**：从自己的素材库做封闭语料合成，结论带频率证据
2. **预注册**：写下预测和判定阈值再开测，防事后找补
3. **单变量对照**：违规版必须是诚实的好版本，不做稻草人
4. **多人设盲审**：评委不知道理论，只回答感知问题；每轮全新实例防记忆污染
5. **识别噪声墙**：批评互相矛盾 + 分数振幅 < 评委差异 → 停止迭代
