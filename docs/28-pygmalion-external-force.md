# 皮格马利翁的雕像：为什么"看"救不活你的项目

> 2026-07-17 · 希腊神话×AI 系列之五（收官）：僵尸项目实锤 + 全系列唯一的 happy ending

## 一个我救不活、也删不掉的 App

我 vibe-code 过一个英语陪练 App。它的出生理由是一个很美的原理：苏格拉底式追问——用提问逼学习者自己生成答案。原理太美了，我想：那就做个工具。

几个月后，它成了我硬盘里的僵尸。我对它的真实感受是一句拧巴的话："不想放弃是因为原理好，不想继续是因为没做出好工具。"我当时把这当成一种情绪，叫它"纠结"。直到被旁人点破才看清：这不是情绪，是个结构错误——**我最珍视的那个抽象原理，正被它最烂的一个载体当人质绑架。**感情认不出看不见的原理，只会黏在看得见的工具上：它有界面、能运行、投入过沉没成本。载体一烂，我连原理一起不敢放手。

期间我做过的所有"抢救"，全是同一个动作：再看一眼，再打磨一轮，再想想加个什么功能。全是凝视。这个项目从没获得过一条不由我产生的输入——没有真实用户，没有别人的眼睛，连"如果要收钱，该收谁的钱"都没被问过。后来拿判定框架逐个跑我的自用工具，翻出 ≥6 条同根记录：**痛驱动开搭，变现逻辑缺席。**

这个形状，奥维德也写过。而且是这个系列五个神话里，唯一一个好结局。

## 雕刻家和他的象牙少女

《变形记》第十卷：雕刻家皮格马利翁雕出一座象牙少女，太美，他爱上了自己的作品。

注意奥维德接下来写了什么：他每天凝视它、跟它说话、给它穿衣，送它贝壳、小鸟、花和琥珀。一个创作者能对作品做的事，他全做了。而象牙还是象牙。**凝视的时长和雕像的生命之间，没有任何函数关系。**

让雕像活过来的是两个动作，都发生在工作室外面：阿芙罗狄忒的节日那天，他离开雕像，去神坛前献祭，说出愿望——他甚至不敢直说"让雕像活"，只敢说"愿我的妻子像那位象牙少女"。回家吻雕像时，指尖下的象牙是温的。

好结局成立的条件刻薄得很清楚：**雕刻是他的，复活不是。**生命来自一个外部力量，且只在他停止凝视、向外开口之后到来。

## 造它的人，中毒最深

工程史给"人对软件动感情"留过一个公开命名。1966 年，MIT 的 Joseph Weizenbaum 写了聊天程序 ELIZA——用模式匹配扮演心理咨询师，没有任何理解能力。让 Weizenbaum 不安的不是程序多强，是人的反应：明知对面是程序，人们仍对它产生真实的情感依恋、向它倾诉；据他记述，他的秘书甚至请他离开房间，好跟 ELIZA"单独谈谈"。这个现象后来叫**伊莱莎效应（ELIZA effect）**：人会向并无理解力的软件，读入理解与感情。

对造物者，这个效应有个加强版：你比任何用户都清楚它背后的原理有多美——软件缺的所有意义，你自己全能补上。皮格马利翁盯着象牙看到的不是象牙，是他刻进去的那个理想；我盯着陪练 App 看到的不是 App，是苏格拉底。凝视越久，投进去的感情越多，就越分不清"它很好"和"我希望它很好"。

（消歧一句：这和"皮格马利翁效应"不是一回事——那是 Rosenthal & Jacobson 1968 年的教师期望研究，讲期望改变他人表现。同名，不同物。）

## 解绑四个动作

1. **出生证解绑（动手前写一行）**：「我真正要的是 [原理X]。这工具只是它的一个载体。工具死了，原理还活在 [现成的承载方式]。」写进 README 的第一天。将来杀它，就不像背叛。
2. **给凝视设预算**：第 N 次说"再打磨一轮再给人看"时停下——打磨不产生外部信息。问一句：这个项目上一次获得一条不由我产生的输入，是什么时候？
3. **问"谁付钱"**：痛存在 ≠ 能变现。"我有这个痛"是需求存在的证据，不是"有人会为它付钱"的证据。该付钱的客户段结构性没钱时，不是放弃——是把收钱对象挪到不缺钱的邻居。
4. **献祭 = 把它带出工作室**：雕像活在皮格马利翁离开之后。给它三个真实用户、一条公开链接、一次别人的眼睛——哪怕它不完美。外力只在你向外开口之后到来。

## 收官：五个神话，一套治理观

这个系列到这里收官。五个神话，是人机协作里五个不同的岗位：

- **桅杆（奥德修斯）＝约束**：规则拦不住临场的自己，清醒时上锁的 hook 才拦得住。
- **水面（纳西索斯与艾可）＝闭环**：出题、执行、判分同体，每一环说真话结论仍可错——在水边放一面别人的镜子。
- **点金术（迈达斯）＝目标**：愿望会被字面执行，写错的目标比没有目标更危险。
- **预言（卡珊德拉）＝警告**：说真话不等于被听见，警告的可信度需要被设计。
- **雕像（皮格马利翁）＝外力**：造是你的事，活不活是外面世界的事。

合成一句话：**给自己上约束，给回路开外部输入，把目标写对，把警告当信号，把生死交给外面的世界。**前四篇讲的都是怎么防止人机协作变糟；只有收官这篇讲怎么让它活——而答案恰好不在人机之间，在人机之外。

## 尾声：好结局为什么给了他

五个神话人物里，皮格马利翁不是最聪明的，也不是最强的。神话把唯一的好结局给他，不是奖励他爱得深——前面四位哪个爱得不深——是奖励他松手松得对：他承认了作品的生命不在自己手里。

凝视是创作的前一半。后一半在工作室外面。雕像最后睁开眼睛，看到的不是雕刻家的凝视——是世界。
