# uu-48 — 在"给谁看/发去哪/什么算完成"定下来之前，别急着优化交付物

> 一场做 AI 黏土科普视频的会话里挖出来的（同日第二条）——从生成到剪辑做了 5+ 版 + 两张封面，最后一条命令才暴露：这视频是要发某社交平台的、面向某语言圈的。

## UU：在交付物的目标（受众 / 渠道 / 完成标准）被定义之前，就投入大量精力优化它的各个维度——优化在真空里进行，没有目标函数可指

- **实锤**：整场从标题卡 → 多场景 → 拼接 → 配音 → 换更自然的配音 → 字幕 → 两版封面 → 封面进视频，做了 5+ 版、两张封面。直到最后一步才暴露它的 destination＝发某社交平台的 build-in-public 帖、受众是某个非英语的语言圈。若这个目标开局就定，好几个选择会不同：**旁白/字幕的语言、时长、开头 hook 的风格、甚至要不要某个版本**。当前用的语言组合对目标受众未必最优——它只是"目标未定"时的默认值。

- **机制 · 概念点名**：**行动偏误（action bias）**——倾向"先做点什么"（不断加功能 / 打磨）而非先停下定义"要什么结果、给谁"。产出感替代了目标定义。〔诚实：action bias 的学术定义偏"宁动勿静"，这里是"先产出、后定目标"，属近亲、不完全等同，只作参照，不硬套术语——心理学术语是编造重灾区，宁可描述机制也不安错名字。〕

- **修法**：动手做任何交付物前，先用一句话填三格——**给谁看＝__ / 发去哪＝__ / 什么算完成＝__**。这一句能倒推语言、时长、格式、开头 hook。本场若开局就填"发某平台 / 某语言圈 / build-in-public"，"用哪种语言配音"那个岔口当场就会被质疑，而不是做到最后才发现选错了默认值。

## 最高杠杆 + 立即可做

把这三格当**任何"帮我产出 X"之前的开场白**：先写「受众 / 渠道 / 完成标准」，三格填不满就别动手——因为填不满，就等于你正在优化一个还没有目标的东西。**你在打磨的每一个维度，都在替一个还没定的目标做假设。**

## 诚实反束

这条有分析者（本文作者，一个 AI 助手）的一半——当用户的请求处于模糊 / 探索态时，助手本该**反向采访、先问"这东西给谁、发哪"**，却没问、跟着一步步执行了。所以这更是**共同盲点**。而且这是同一场里第二条都带"分析者也有责任"的观察——说明做这份分析的验证器，对自己那一半的觉察本身也可能不足。单场一次观察不是判决；分析者是没被验证的验证器，请打折收下。
